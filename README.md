# Conway's Game of Life

## What is it?
An implementation of [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) in C using `ncurses` where each new game starts at random.

## Controls
* `Q` to quit
* `R` to restart
* `W` to speed up
* `S` to slow down
* `D` to set the default speed
* ` ` to pause/unpause
